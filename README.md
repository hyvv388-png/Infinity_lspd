## ğŸ’» Customization

### Change Content
- Edit HTML files to modify text, add/remove pages
- Update data arrays in `js/main.js` for members, vehicles, achievements, etc.

### Change Colors
- Edit CSS variables in `css/style.css`:
  - `--color-dark-navy`: #0D1B2A
  - `--color-police-blue`: #1B263B
  - `--color-accent-blue`: #0EA5E9

### Change Fonts
- Update `--font-primary` in `css/style.css`
- Add Google Fonts link in HTML `<head>` section

### Add New Pages
1. Create new HTML file with proper structure
2. Add navigation link to all pages
3. Add initialization function in `js/main.js`
4. Add CSS styles in `css/style.css`
